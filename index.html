Address Plotter

We originally wrote this code so we could see where people lived relative to our church. We then thought that other people might want to plot addresses around a single address. The code below will allow you to do the same thing using the Google Maps API.

Below the following addresses (red map tags) around "15980 Blossom Hill Road, Los Gatos, CA 95032" (orange map tag):

  • 15980 Los Gatos Boulevard Los Gatos, CA 95032-3424
  • 15466 Los Gatos Boulevard Los Gatos, CA 95032-2551

First input an address to plot around:

Then put a list of addresses you would like placed on the map. After each address press the return key to ensure that a new line is created. The script is separating each address by a carriage return (\n). For example:

15980 Los Gatos Boulevard Los Gatos, CA 95032-3424
15466 Los Gatos Boulevard Los Gatos, CA 95032-2551

Once you hit submit your center address will be an orange map marker and then your other address will all be red markers.
